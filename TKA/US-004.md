# Historia de Usuario US-004

## Registrar residuos por colegio
**Como** personal de un colegio registrado
**Quiero** registrar nuevos residuos vinculados a mi centro educativo usando los campos definidos en tables.md
**Para** llevar un control preciso de la generación de residuos.

## Criterios de Aceptación

- Existe el endpoint `POST /colegios/{colegio_id}/residuos`
- Se usan estrictamente los campos de `tables.md`:
    - `tipo`: Enum (plastico, papel, vidrio, organico, electronico)
    - `peso_kg`: Float (> 0)
    - `volumen_litros`: Float (> 0)
    - `aula`: String
- El sistema asigna automáticamente:
    - `colegio_id`: Desde la URL
    - `estado`: "pendiente" (según tables.md)
    - `fecha_registro`: Timestamp actual
- Respuesta: `201 Created` con el objeto completo.
