# Historia de Usuario US-010

## Tests para la Gestión de Colegios
**Como** desarrollador backend
**Quiero** tener pruebas automatizadas para la creación y consulta de colegios
**Para** asegurar la integridad de los datos de los centros educativos.

## Criterios de Aceptación

- Se incluye un test de integración para el registro exitoso de un colegio.
- Se incluye un test para validar que no se pueden crear colegios con campos obligatorios vacíos.
- Se incluye un test para verificar la recuperación de la lista completa de colegios.
- Se incluye un test para la recuperación de un colegio por ID.
- Mantenimiento de la cobertura de código por encima del 75%.

## Tareas

| Código | Nombre |
|--------|--------|
| TK-010-01 | Crear `tests/test_colegios.py` con escenarios de éxito y error |
| TK-010-02 | Mockear o usar la base de datos de test definida en `conftest.py` |
| TK-010-03 | Ejecutar tests con `pytest` y generar reporte |
