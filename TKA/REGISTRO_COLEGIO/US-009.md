# Historia de Usuario US-009

## Consulta de Colegios Registrados
**Como** administrador o usuario del sistema
**Quiero** visualizar la lista de todos los colegios registrados o buscar uno por su ID
**Para** gestionar los centros asociados a la red de reciclaje.

## Criterios de Aceptación

- Existe el endpoint `GET /colegios` que devuelve la lista de todos los colegios.
- Existe el endpoint `GET /colegios/{colegio_id}` que devuelve la información de un centro específico.
- La respuesta utiliza el esquema `ColegioResponse` definido en los schemas.
- Si se busca un colegio por ID y no existe, devuelve un error `404 Not Found`.

## Tareas

| Código | Nombre |

| TK-009-02 | Implementar los endpoints GET en `routes/colegio.py` |
| TK-009-03 | Validar que la información devuelta sea coherente con lo guardado en la DB |
| TK-009-04 | Probar el manejo de errores al buscar un ID inexistente |
