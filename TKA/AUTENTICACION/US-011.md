# Historia de Usuario US-011

## Credenciales de Acceso para Colegios
**Como** administrador del sistema
**Quiero** que los colegios tengan un correo electrónico único y una contraseña segura
**Para** que puedan autenticarse y proteger su información.

## Criterios de Aceptación

- La tabla `colegios` debe incluir los campos:
    - `email`: VARCHAR(150), único, no nulo.
    - `password`: VARCHAR(255), no nulo (almacenará el hash).
- Durante el registro (`POST /colegios`), estos campos deben ser obligatorios.
- La contraseña debe almacenarse utilizando un algoritmo de hash seguro (ej: bcrypt).
- El sistema debe validar que el formato del email sea correcto.
