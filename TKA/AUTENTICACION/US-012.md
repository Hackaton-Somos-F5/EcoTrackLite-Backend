# Historia de Usuario US-012

## Autenticación de Colegios (Login)
**Como** gestor de un colegio registrado
**Quiero** iniciar sesión con mi correo y contraseña
**Para** acceder a las funcionalidades de gestión de residuos.

## Criterios de Aceptación

- Existe el endpoint `POST /auth/login`.
- El endpoint recibe un JSON con `email` y `password`.
- Si las credenciales son correctas, devuelve la información básica del colegio y un token (opcional por ahora, puede ser un mensaje de éxito).
- Si las credenciales son incorrectas (email no encontrado o contraseña errónea), devuelve un error `401 Unauthorized`.
- Se implementa la verificación del hash de la contraseña.
