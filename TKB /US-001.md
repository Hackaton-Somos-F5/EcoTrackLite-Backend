# HU-B01 · Resumen de volumen por categoría

**Como** usuario del dashboard escolar  
**Quiero** ver el volumen total acumulado agrupado por tipo  
**Para que** el frontend pueda mostrar un gráfico de barras con el impacto por categoría.

---

##  Criterios de Aceptación

- [ ] **Endpoint:** `GET /residuos/resumen` devuelve un objeto con cada tipo y su volumen total.
- [ ] **Valores nulos:** Los tipos sin residuos registrados deben devolver `0`.
- [ ] **Rendimiento:** El cálculo debe usar agregación en base de datos (`SUM`, `GROUP BY`), no procesamiento en memoria con Python.
- [ ] **Formato de respuesta esperado:**
  ```json
  { 
    "plastico": 120.5, 
    "papel": 80.0, 
    "organico": 200.3 
  }
